gdb init script

Created: Feb 25, 2025 · 1 min read

.gdbinit 能够将gdb基本命令添加到文件中以避免每次启动gdb 进行 debug 时候输入他们，例如

b main
r
b main.c:56
c

这样就可以避免重复性的劳动。

通过也可以在 .gdbinit 中 自定义命令：

define <command name>
    <code>
end

在 gdb 中可以直接调用通过 <command name> 来调用该自定义命令。同时也支持参数列表，在脚本中 使用形如 $arg1，$arg2，….，来访问参数列表，在gdb中就可以这样调用：

> <command name> arg1 arg2 arg3 ...

在 .gdbinit 脚本中，可以使用 $<variable_name> 变量的名字。

.gdbinit 也支持 if 条件判断：

if <condition>
    <code>
else
    <code>
end

以及 while 循环：

while <condition>
    <code>
end

打印在 .gdbinit 中也很简单：

printf "<format>", arg0, arg1, ...
